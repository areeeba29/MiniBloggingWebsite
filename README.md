# 📝 Mini Blogging Website

A simple **web-based blogging platform** developed as part of the **Software Construction & Development (SCD)** course.  
The project demonstrates **OOP concepts, UML diagrams, frontend & backend integration, database connectivity, and GitHub version control**.

---

## 🚀 Features

- **User Management**
  - Register / Sign Up
  - Login & Logout
  - Update Profile
  - User Dashboard (list of user’s blogs)

- **Blog Management**
  - Create Blog (title, content, category)
  - Edit Blog (only by author)
  - Delete Blog (confirmation required)
  - View Blogs (all users)

- **Search & Categories**
  - Search blogs by title or keyword
  - Categorize blogs (e.g., Tech, Education, Lifestyle)

- **Guest Access**
  - Browse and search posts without logging in

---

## 🛠️ Tools & Technologies

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Java (Servlets, OOP principles)  
- **Database:** MySQL (via JDBC)  
- **Modeling:** UML diagrams (Use Case, Class, Sequence)  
- **Version Control:** Git & GitHub
